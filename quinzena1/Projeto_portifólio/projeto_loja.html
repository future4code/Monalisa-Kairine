Projeto Loja

O objetivo deste projeto foi construir um sistema interno de uma loja usando inicialmente somente HTML. Na etapa seguinte o projeto foi expandido e foi adicionado um arquivo CSS em todas as páginas.
O sistema da loja é constituído de 4 páginas, sendo elas:

  • Página inicial
  • Controle de Estoque
  • Adicionar produtos
  • Funcionários

As principais ferramentas utilizadas foram:

  • HTML
  • CSS
